========================================================================
要件・仕様定義書：ワークバータイマー（Web & デスクトップ・ハイブリッド版）
========================================================================

1. 開発概要
------------------------------------------------------------------------
本プロジェクトは、キングジム社製の「ビジュアルバータイマー」の持つ「20個の目盛り（バー）の増減による直感的な時間管理」という優れたコンセプトをベースに、WebブラウザおよびPCデスクトップ上で動作するクロスプラットフォーム対応のタイマーアプリとして開発するものである。

2. プラットフォームおよび技術スタック
------------------------------------------------------------------------
* フロントエンド：TypeScript, HTML5, CSS3 (Tailwind CSS 等の利用を推奨)
* デスクトップ版：Neutralinojs (軽量・低メモリ消費を重視)
* ブラウザ版：モダンブラウザ対応 (レスポンシブWebデザイン)
* ホスティング・検証：GitHub Pages (ブラウザ版公開用), GitHub Actions (CI/CD)

3. システムアーキテクチャ
------------------------------------------------------------------------
拡張性とポータビリティを担保するため、ドメイン分離（オニオンアーキテクチャ思想）を採用する。

* ドメイン層（Core Domain）：
  - 純粋なTypeScriptで記述し、システム日時（Date型）や特定のプラットフォームAPI（window, navigator, Neutralino等）に一切依存しない。
  - 時間の経過は外部から「デルタタイム（経過秒数）」として引数で注入（DI）される。
  - 状態遷移（IDLE, RUNNING, PAUSED, TIME_UP）および20個のバーの点灯数計算ロジックをここに集約する。

* インフラストラクチャ層 / アダプター層：
  - ドメイン層を各環境（Neutralinojs / ブラウザ）に適合させる。
  - ユニットテストの実行環境（GitHub Actions上の高速シミュレーション）もここを通じて行う。

4. 主要機能要件
------------------------------------------------------------------------
### 4.1. タイマー表現とコア機能
* 20段階ビジュアルバー：残り時間に応じて20個のブロックが動的に減少/増加。
* 4色カラー目盛り：残り時間（パーセンテージ）に応じて、バーの色が緑→黄→橙→赤へとグラフィカルに変化する（VBT20仕様の再現）。
* カウントダウン / カウントアップ計測：通常のタイマー機能に加え、超過時間や累積時間の計測に対応。
* リピート計測（ポモドーロモード）：「作業時間」と「休憩時間」を交互に最大99回まで自動で繰り返しカウントする。

### 4.2. デスクトップ版（Neutralinojs）固有機能
* 横長枠なしウィンドウ（Frameless / Borderless）：タイトルバーや外枠を排除し、実機のようなガジェット感を再現。
* 常に最前面表示（Always on Top）：他の作業ウィンドウに隠れることなく、常に視界の端に配置可能。
* ウィンドウドラッグ移動：枠なし状態でも、画面上の任意の場所へマウスドラッグで配置を変更可能。
* タスクバー連携：タイムアップ時にタスクバーやDockのアプリアイコンを点滅・バウンドさせユーザーの注意を引く（Neutralino.window.attention）。

### 4.3. ブラウザ版固有機能
* フルスクリーン表示（Fullscreen API）：スマホやPCの画面全体にタイマーを拡大表示。
* スリープ防止機能（Screen Wake Lock API）：全画面表示中、スマホやタブレット等の端末が自動でスリープ・ロック状態になるのを防止（常時点灯の据え置きタイマー化）。

### 4.4. 操作性・UI拡張（デジタルならではの機能）
* マウスホイール連携：実機のダイヤル操作を模し、タイマー上でホイールをスクロールすることで1分/10秒単位の時間設定が可能。
* キーボードショートカット：Spaceでのスタート/ストップ、Escでのリセット等。
* 消音・視覚通知：タイムアップ時は音を鳴らさず、OS標準の通知（トースト通知）および、画面全体がバックライトのようにチカチカと点滅するCSSアニメーション（フラッシュ通知）で知らせる。

5. 開発およびデプロイフロー
------------------------------------------------------------------------
1. 純粋なTypeScriptによるドメイン層（Timer.ts）の作成およびユニットテストの構築。
2. ローカルブラウザ環境（HTML/CSS/TS）でのプロトタイプ作成（UIとタイマーロジックの結合）。
3. GitHubリポジトリへのプッシュをトリガーとしたGitHub Actionsによる自動テスト。
4. テスト通過後、自動的にGitHub Pagesへブラウザ版をデプロイ。
5. ビルド成果物をNeutralinojsプロジェクト（app.jsonの設定適用）へ流し込み、デスクトップアプリ（.exe / .app）としてパッケージング。
